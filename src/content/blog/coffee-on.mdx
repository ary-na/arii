customers can browse and order drinks, baristas manage incoming orders.

## Tech Stack

```bash
# frontend
lit.js | parcel | sass | gsap
```

```bash
# backend
node.js | express.js | mongodb | mongoose | jwt
```

## Key Features

- User authentication (JWT)
- Browse and order drinks
- Shopping cart
- Favourites system

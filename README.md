# arian.dev

Personal blog where I write about what I'm learning, what I'm building, and the tools and systems I work with.

## stack

- [Astro](https://astro.build/) — static site generator
- [Tailwind CSS v4](https://tailwindcss.com/) — styling
- [TypeScript](https://www.typescriptlang.org/) — type safety
- [MDX](https://mdxjs.com/) — markdown with components

## hosting

Statically hosted on [AWS S3](https://aws.amazon.com/s3/) with [Amazon Route 53](https://aws.amazon.com/route53/) for DNS management.

## fonts

- **Bytesized** — display / headings
- **DM Sans** — body text
- **DM Mono** — code blocks

## features

- retro terminal aesthetic
- dynamic category system
- dark mode via `prefers-color-scheme`
- syntax highlighted code blocks
- responsive — mobile, tablet, desktop layouts
- SEO meta tags and open graph support
- RSS feed
- sitemap

## development
```bash
npm install
npm run dev
```

## build
```bash
npm run build
npm run preview
```
